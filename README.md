# SmartEats - Landing Page
SmartEats es una landing page interactiva creada con TailwindCSS y JavaScript, cuyo objetivo es presentar una propuesta que ayuda a estudiantes y jóvenes profesionales a reducir la fatiga y mejorar su rendimiento a través de mejores hábitos alimenticios.

La página busca resolver dolores comunes como:

* Bajones de energía durante estudio/trabajo.

* Falta de tiempo para comer bien.

* Elecciones rápidas poco saludables.

* Desorden en horarios de comida e hidratación.

SmartEats ofrece:

* Guía gratuita “7 días de energía” (suscripción por email).

* Recetas rápidas y snacks inteligentes.

* Tips prácticos para hidratación, horarios y balance de macronutrientes.
